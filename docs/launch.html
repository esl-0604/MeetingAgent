브라우저에서 "이 사이트가 Meeting Agent를 열려고 합니다" 같은 알림이 뜨면 "열기" 를 누르세요. "항상 허용"을 체크하면 다음부터는 자동으로 열립니다.

⚠️ Meeting Agent 가 아직 설치되어 있지 않습니다

처음이신가 봐요. 아래 순서대로 1분만 투자하시면 이후엔 바로 실행됩니다.

  1. Installer 다운로드
  2. 받은 MeetingAgent_installer.zip 을 원하는 폴더에 압축 해제
  3. 풀린 폴더의 install.bat 을 더블클릭 (Windows SmartScreen 경고 시 추가 정보 → 실행)
  4. 설치가 끝나면 이 페이지의 링크를 다시 클릭

관리자 권한 없이 현재 사용자 영역(%LOCALAPPDATA%\MeetingAgent)에 설치됩니다.

요청된 동작: show
프로토콜: meetingagent://show
